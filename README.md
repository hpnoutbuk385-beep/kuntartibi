# 🚀 Kundalik Tartib - Kelajagingni Esla!

Har kuni, har soatda sizni tartibingizni bajarishga undaydigan, motivatsiya beradigan va ogohlantiradigan oddiy veb-ilova.

Asosiy g‘oya:  
“Buni qilmasang — kelajaging bo‘lmaydi!” ruhida kuchli eslatmalar.

---

## ✨ Imkoniyatlar

- Har soat (yoki 30 daqiqada) avtomatik eslatmalar
- Telefon bildirishnomalari (Push Notification)
- PWA — telefoningizga alohida ilova sifatida o‘rnatish mumkin
- Internet o‘chsa ham asosiy funksiyalar ishlaydi
- Vazifalarni bajarilgan deb belgilash
- Motivatsion xabarlar
- O‘zbek tilida
- Zamonaviy va chiroyli dizayn (Dark Mode)

---

## 📱 Qanday ishlatish

### 1. Tez sinash (eng oson)

1. Ushbu papkadagi `index.html` faylini brauzerda oching.
2. Agar telefonda bo'lsangiz, Chrome brauzerida ochib, menyudan “Bosh ekranga qo‘shish” (Add to Home Screen) tugmasini bosing.
3. Ilovani oching va bildirishnomalarga ruxsat bering.

### 2. Dasturni joylash (Hosting)

Ilovani internetga chiqarish uchun fayllarni quyidagi bepul xizmatlardan biriga yuklashingiz mumkin:
- **Vercel** (vercel.com)
- **Netlify** (netlify.com)
- **GitHub Pages** (github.com)

Hech qanday uchinchi tomon xizmati (OneSignal kabi) talab qilinmaydi. Ilova brauzerning o'zida native push bildirishnomalarini ishlatadi.
